# aoc-2025-ruby

## Installation

1. Install bundler if you haven't already:

   ```bash
   gem install bundler
   ```

2. Install dependencies using bundler:
   ```bash
   bundle install
   ```

## Configuration

1. Create a `.env` file in the root directory.
2. Add your Advent of Code session cookie to the `.env` file:
   ```
   SESSION=your_session_cookie_here
   ```
   You can find your session cookie by inspecting your browser's cookies while logged into adventofcode.com.

## Usage

Run the individual solution files using the `ruby` command:

```bash
ruby aoc1-1.rb
ruby aoc1-2.rb
# etc...
```
